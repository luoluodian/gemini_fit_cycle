# 饮食记录H5应用交互设计文档

## 应用概述
一个功能完整的饮食记录与管理H5应用，帮助用户追踪每日饮食、制定饮食计划、管理食材库。

## 核心交互功能

### 1. 记录(今日)页面交互
- **日期导航**：点击左右箭头切换日期，点击日期显示日历选择器
- **目标概览**：动态显示当日热量和营养素目标进度
- **餐次记录**：
  - 点击"+添加食物"打开食材选择弹层
  - 点击"按计划记录本餐"自动填充预设餐单
  - 点击已记录食物可编辑数量或删除
- **进度可视化**：实时更新热量和营养素进度条
- **建议反馈**：根据摄入情况显示个性化建议

### 2. 计划管理页面交互
- **计划列表**：
  - 点击计划卡片进入详情页
  - 长按或点击"..."显示操作菜单（编辑、激活、删除等）
  - 滑动切换"进行中"和"已归档"计划
- **新建/编辑计划**：
  - 多步骤表单：基本信息 → 目标配置 → 菜谱配置
  - 日期选择器设置计划起止时间
  - 切换"统一目标"和"按天设置"模式
- **每日配置**：
  - 日历视图选择配置日期
  - 表单设置当日热量和营养素目标
  - 餐单配置界面添加/编辑每餐食物

### 3. 食材库页面交互
- **搜索功能**：实时搜索食材名称
- **分类浏览**：切换"全部"/"系统食材"/"我的食材"标签
- **食材详情**：
  - 显示营养成分信息
  - 编辑/删除自定义食材
  - 设为常用食材
- **新建食材**：表单添加自定义食材信息

### 4. 个人中心页面交互
- **基础信息设置**：编辑身高、体重、活动水平等
- **目标建议工具**：输入个人信息获取BMR和TDEE计算
- **统计图表**：显示最近7天摄入趋势图表
- **计划导入**：通过分享码导入他人计划

## 通用交互组件

### 1. 日期选择器
- 日历网格显示
- 支持月份切换
- 高亮显示有记录的日期
- 快速跳转到今天

### 2. 食材选择弹层
- 顶部搜索框
- 最近使用食材快捷选择
- 分类标签筛选
- 数量输入和确认

### 3. 数量编辑弹层
- 数字输入框
- 常用份量快捷按钮
- 单位选择器
- 确认和取消操作

### 4. 进度条组件
- 动态数值更新
- 颜色渐变表示完成度
- 悬停显示详细信息

## 数据流设计
- 本地存储用户数据
- 实时计算营养素统计
- 跨页面数据同步
- 计划分享和导入功能

- 滑动操作支持
- 长按菜单
- 下拉刷新
- 无限滚动加载
