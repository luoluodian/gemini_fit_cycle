# 项目技术栈文档

## 项目基础架构（基本框架已经搭建完成）

- **后端** code_src/fit_cycle_app
- **前端** code_src/fit_cycle_web

## 项目规则

- 目前不考虑接入外部的 api（只完成内循环）；但是 ai 模块可以保留，但是不启用

## 后端技术栈

### 核心框架

- **NestJS** (v11.1.9) - 基于 Node.js 的渐进式框架，采用 TypeScript 开发
- **TypeScript** (v5.9.3) - 提供类型安全的 JavaScript 超集

### 数据库

- **MySQL** (v3.15.3) - 关系型数据库，通过 mysql2 驱动连接
- **TypeORM** (v0.3.27) - ORM 框架，支持实体关系映射
- **Redis** (ioredis v5.8.2) - 用于缓存和会话存储

### 认证授权

- **JWT** (@nestjs/jwt v11.0.1) - JSON Web Token 认证
- **Passport** (@nestjs/passport v11.0.5) - Node.js 认证中间件
- **passport-jwt** (v4.0.1) - JWT 策略支持

### 日志系统

- **Winston** (v3.18.3) - 多传输日志库
- **nest-winston** (v1.10.2) - NestJS Winston 集成
- **winston-daily-rotate-file** (v5.0.0) - 日志文件轮转

### 数据验证与转换

- **class-validator** (v0.14.2) - 基于装饰器的验证
- **class-transformer** (v0.5.1) - 对象转换工具

### 开发工具

- **ESLint** (v8.0.0) - 代码质量检查
- **Prettier** (v3.0.0) - 代码格式化
- **Jest** (v29.5.0) - 测试框架
- **TypeScript ESLint** - TypeScript 代码检查

### 部署与容器化

- **Docker** - 容器化部署
- **Node.js** (v22-alpine) - 运行时环境
- **Yarn** - 包管理器

## 前端技术栈

### 核心框架

- **Taro** (v4.1.9) - 多端统一开发框架
- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全的 JavaScript 超集

### 样式方案

- **Sass** - CSS 预处理器
- **Tailwind CSS** - 实用优先的 CSS 框架

### 状态管理

- **Pinia** - Vue 3 官方推荐的状态管理库

### UI 组件库

- **NutUI Taro** - 基于 Taro 的移动端组件库

### 构建工具

- **Vite** - 下一代前端构建工具

### 支持平台

- 微信小程序
- H5

## 开发环境要求

### 后端

- Node.js >= 18.0.0
- MySQL >= 8.0
- Redis >= 6.0
- Yarn >= 1.22.0

### 前端

- Node.js >= 18.0.0
- 微信开发者工具（小程序开发）
- Yarn >= 1.22.0

### 开发工具

- Docker（可选，用于本地开发环境）
- VS Code（推荐编辑器）

### 服务器

后台： 一台 2c2g 服务器
前端： 微信小程序
