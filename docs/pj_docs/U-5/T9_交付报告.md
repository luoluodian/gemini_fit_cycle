## 1. 交付概览

### 1.1 功能描述
实现了用户个人中心页面，包含以下核心模块：
-   **Profile Header**: 展示用户头像、昵称及坚持天数。
-   **Current Plan**: 展示当前进行中的饮食计划状态（周数、剩余天数），提供“查看”和“编辑”入口。
-   **Quick Stats**: 展示“坚持天数”和“完成计划数”核心指标。
-   **Weekly Chart**: 可视化展示本周热量摄入趋势及达标情况。
-   **Menu Grid**: 提供健康工具（BMR计算、体重记录、营养指南）、设置及导入计划入口。
