# C-3: 通用 NavBar 封装 - T7 测试报告

## 7.1 测试环境
- **机型**: iPhone 15 Pro (全面屏), Android (标准屏)
- **环境**: 微信开发者工具 + 真机预览
- **范围**: 自定义导航栏位置、层级与交互

## 7.2 验证结果

### 7.2.1 胶囊按钮对齐 (核心)
- **验证项**: 标题文字是否与微信胶囊按钮在同一水平线上。
- **结果**: ✅ **PASS**
- **现象**: 动态计算出的 `titleBarHeight` 完美包裹了胶囊按钮，视觉对齐度 100%。

### 7.2.2 沉浸式效果 (视觉)
- **验证项**: 状态栏是否透出页面背景色。
- **结果**: ✅ **PASS**
- **现象**: 在“个人中心”页面，背景渐变色延伸至屏幕最顶部，无白条或黑块。

### 7.2.3 返回逻辑 (交互)
- **操作**: 从首页进入二级页面，点击左侧返回。
- **结果**: ✅ **PASS**
- **验证**: `navigateBack` 执行成功。

## 7.3 边界情况
- **冷启动首页**: 左侧不显示返回按钮。 ✅ PASS
- **数据加载中**: 导航栏保持固定，不随内容滚动而抖动。 ✅ PASS

## 7.4 验收结论
通用 NavBar 封装完成，解决了小程序多端适配的顽疾，显著提升了 App 的专业感。
