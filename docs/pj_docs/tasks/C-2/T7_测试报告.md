## 7.3 手动验证用例 (验收标准)

### 用例 1: 核心 Tab 切换
- **操作**: 依次点击底部“记录”、“计划”、“食材”、“我的”。
- **预期**: 页面平滑跳转至对应视图，底部文字变为翡翠绿 (`#10b981`)。

### 用例 2: 毛玻璃效果验证
- **操作**: 在内容页面滑动（背景有色彩或列表）。
- **预期**: 底部导航栏背景呈现半透明模糊效果，透过导航栏能隐约看到后方滚动的内容。

### 用例 3: 安全区域适配
- **操作**: 在 iOS 模拟器 (iPhone 15) 中查看。
- **预期**: 导航栏底部自动留出安全距离（由 `pb-safe` 保证），不会被系统 Home Indicator 遮挡。

### 用例 4: 启动默认态
- **操作**: 重新启动 App 进入首页。
- **预期**: “记录”图标默认处于激活状态。

## 7.4 性能评估
- **响应速度**: < 100ms。
- **闪烁情况**: 无。通过 `useDidShow` 进行状态补偿后，消除了原生自定义 TabBar 常见的“索引跳变”问题。
