# F-2: 食物搜索接口 - T5 分析文档

## 5.1 模块关联性审计
- **核心功能**: 提供基于关键词的食物搜索及基于分类的过滤。
- **关联组件**: 前端搜索栏 (`SearchBar.vue`) 和 分类标签 (`CategoryTabs.vue`)。

## 5.2 需求实现点
- **模糊查询**: 必须支持对 `name` 字段的 `LIKE` 匹配。
- **分类筛选**: 必须支持对 `category` 字段的精确匹配。
- **性能**: 搜索作为高频操作，需确保数据库索引命中。

## 5.3 技术实现方案
- **逻辑方案**: 通过 `where` 语句动态组合 `name` 和 `category` 条件。
- **代码复用**: 复用 `FoodService.findFood` 方法，但在本任务中重点验证搜索条件的生效性。

## 5.4 验收标准
- [ ] 输入关键词能返回匹配的食物。
- [ ] 切换分类能返回对应类别的食物。
