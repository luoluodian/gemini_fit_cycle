## 5.3 技术落地计划

### 1. 插入方式
采用 SQL `INSERT INTO` 批量脚本。这种方式比代码脚本更直接，且易于在数据库管理工具中执行。

### 2. 覆盖范围
初期导入 20-30 种核心食材，涵盖基础蛋白质、优质碳水、常见蔬果及油脂。

### 3. 数据校验
确保 `calories_per_100g` 符合公式近似：`蛋白质*4 + 脂肪*9 + 碳水*4`。

## 5.4 验收标准 (T7 预案)

- [ ] 数据字段完整，无 NULL 值报错。
- [ ] 升糖指数 (GI) 数据已正确填充。
