- `fit_cycle_web/src/components/login/LoginButton.vue`:
    - 添加了“先体验一下” (先体验一下) 游客模式按钮。
    - 添加了交互反馈动画及样式适配。
- `fit_cycle_web/src/pages/login/index.vue`:
    - 整合了所有登录组件 (`LogoHeader`, `WelcomeCard`, `LoginButton`, `PrivacyModal`)。
    - 引入 `animejs` (v4 animate/stagger) 实现了页面元素的阶梯式入场动画。
    - 实现了真实的 `handleWechatLogin` 流程，调用 `Taro.login()` 获取 code 并通过 `userStore.login` 进行认证。
    - 实现了 `handleGuestMode` (游客模式)，通过 `demoMode` 本地存储进行标记。
- `fit_cycle_web/src/pages/login/index.scss`:
    - 完善了毛玻璃卡片 (`.glass-card`)、微信登录按钮 (`.wechat-btn`) 以及浮动动画样式。
    - 适配了 Tailwind CSS 类名与 Sass 变量。
