# 系统架构逻辑映射表 (Architecture Map)

本文档旨在建立前端页面操作、后端服务逻辑与数据库模型之间的清晰映射关系，作为系统设计的核心参考。

| 页面/操作 (Page/Operation) | 前端组件/逻辑 (Frontend Component/Logic) | 后端服务 (Backend Service) | 涉及数据表 (Affected DB Tables) |
| :--- | :--- | :--- | :--- |
| **用户登录** | `/pages/login/index` -> `LoginButton` | `AuthService.login()` | `users` (查询/创建) |
| **查看每日饮食** | `/pages/index/index` -> `onShow` | `DietLogService.findByDate()` | `diet_logs`, `foods`, `users` (查询) |
| **添加食物记录** | `MealCard` -> "添加食物" -> `FoodSelectionModal` | `DietLogService.createLog()` | `diet_logs` (创建) |
| **搜索食物** | `FoodSelectionModal` -> `SearchInput` | `FoodService.search()` | `foods` (查询) |
| **切换日期** | `DateNavigation` -> "箭头按钮" | `DietLogService.findByDate()` | `diet_logs`, `foods`, `users` (查询) |
| **查看所有计划** | `/pages/plan/index` -> `onShow` | `PlanService.listByUser()` | `user_plans`, `plan_templates` (查询) |
| **激活某个计划** | `PlanCard` -> "激活"按钮 | `PlanService.activatePlan()` | `user_plans` (更新) |
| **创建自定义计划** | `PlanHeader` -> "创建计划" | `PlanService.createCustomPlan()` | `plan_templates`, `user_plans` (创建) |
| **导入他人计划** | `PlanHeader` -> "导入计划" | `ShareService.importPlan()` | `user_plans` (创建) |
